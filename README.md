Ürün Açıklaması

• Bebeğin aylık olarak duyusal ve fiziksel gelişimi hakkında bilgiler verme 
• Not defteri özelliği ile bebeğin özel anlarını kaydetme
• Aşı takvimi 
• Bakım tavsiyeleri
